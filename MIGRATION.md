# Migration Guide

## v1.0.0 (Initial Release)

Этот релиз представляет собой первую стабильную версию интеграции Chat Frontend и n8n.

### Миграция со статической HTML версии

Если ранее вы использовали просто HTML файлы или старый метод проксирования:

1. **Структура проекта**:
    - Все статические файлы (`index.html`, `style.css`, `chat.js`) перемещены в папку `public/`.
    - Корень проекта теперь содержит конфигурацию Node.js (`package.json`) и конфиги туннелей.

2. **Запуск**:
    - ПРЕДЫДУЩИЙ МЕТОД: Открытие файла в браузере или Live Server.
    - НОВЫЙ МЕТОД: `npm run dev` (для правильной работы модулей и CORS).

3. **n8n Интеграция**:
    - Webhook URL теперь стандартизирован как `https://n8n.mup.me/webhook/chat`.
    - Убедитесь, что ваш n8n workflow слушает именно этот путь и метод POST.

4. **Туннели**:
    - Введены раздельные конфиги для n8n и чата. Старые одиночные команды `cloudflared tunnel --url ...` заменены на файлы конфигурации `config.yml` и `n8n-config.yml`.
