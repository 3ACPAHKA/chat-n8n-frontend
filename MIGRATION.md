## v0.3.0 (Redesign & Stable Sync)

Этот релиз полностью обновляет UI и синхронизирует протоколы обмена данными.

### Основные изменения

1. **Новый Интерфейс (UI/UX)**:
    - Добавлен **Landing Page** (стартовый экран).
    - Переход на **Strict Dark Mode** (светлая тема полностью удалена).
    - Кастомизация кнопки "Отправить": новый 3D-дизайн без текстовой надписи.
    - Реализовано копирование текста сообщения по клику с динамическим уведомлением (Toast).

2. **Технические изменения (Frontend)**:
    - Кэш-бастинг (Query params `?v=X`) теперь используется для всех файлов в `public/`.
    - Очистка старых процессов Node.js при старте (`kill_old_processes.ps1`).
    - Использование нативного `res.text()` вместо JSON-парсинга для ответов бота.

3. **n8n Интеграция**:
    - Переход на формат **Plain Text** для ответов Webhook. n8n теперь возвращает сырой текст от AI без обертки в JSON объекты.

### Рекомендации по обновлению

- Если вы используете старый n8n workflow, обновите узел **Respond to Webhook**, установив `Respond With: Text` и передавая `{{ $json.answer }}` напрямую (без скобок/кавычек).
- Очистите кэш браузера или используйте актуальную версию URL с `?v=19`.
